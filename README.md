# ğŸ“– [Catalog of Things] <a name="about-project"></a>

**[Catalog of things]** is a ruby application that allows users to create different items such as music albums, games, movies and books. The data is stored in JSON files. It also includes a schema file with tables of the classes in the app.

## ğŸ›  Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- Languages: Ruby, SQL
- Frameworks: Rspec
- Technologies Used: GIT, GITHUB, LINTERS

<!-- Features -->

### Key Features <a name="key-features"></a>

- Add books, music albums and games
- List all books, labels, music albums, genres, games & authors
- Data preserved in json file
- Schema file with tables of the classes in the app

## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 ruby installed
```

### Setup

Clone this repository to your desired folder:

```sh
  git https://github.com/ambrose-kibet/Catalog-of-my-things.git
  cd Catalog-of-my-things
```

### Install

Install this project with:

```sh
  cd Catalog-of-my-things
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  ruby main.rb
```

### Run tests

To run tests, run the following command:

```sh
  rspec spec spec\
```
